#### 说明

> 本章节将介绍表格对象的基本用法

---

#### 对象的创建

```
// 表格
XEasyPdfTemplateTable table = XEasyPdfTemplateHandler.Table.build();
// 表头
XEasyPdfTemplateTableHeader header = XEasyPdfTemplateHandler.Table.Header.build();
// 表尾
XEasyPdfTemplateTableFooter footer = XEasyPdfTemplateHandler.Table.Footer.build();
// 表格体
XEasyPdfTemplateTableBody body = XEasyPdfTemplateHandler.Table.Body.build();
// 表格行
XEasyPdfTemplateTableRow row = XEasyPdfTemplateHandler.Table.Row.build();
// 表格单元格
XEasyPdfTemplateTableCell cell = XEasyPdfTemplateHandler.Table.Cell.build();
```

#### 示例

> ##### 对象模式 java 代码
```
// 定义输出路径
String outputPath = "E:\\pdf\\test\\fo\\test.pdf";
// 创建文档
XEasyPdfTemplateDocument document = XEasyPdfTemplateHandler.Document.build();
// 创建页面
XEasyPdfTemplatePage page = XEasyPdfTemplateHandler.Page.build();
// 创建表格
XEasyPdfTemplateTable table = XEasyPdfTemplateHandler.Table.build();
// 创建表格体
XEasyPdfTemplateTableBody tableBody = XEasyPdfTemplateHandler.Table.Body.build();
// 循环创建表格行
for (int i = 1; i < 4; i++) {
    // 创建表格行
    XEasyPdfTemplateTableRow tableRow = XEasyPdfTemplateHandler.Table.Row.build();
    // 循环创建表格单元格
    for (int j = 1; j < 4; j++) {
        // 创建表格单元格
        XEasyPdfTemplateTableCell tableCell = XEasyPdfTemplateHandler.Table.Cell.build().setBorderStyle("solid");
        // 创建文本
        XEasyPdfTemplateText text = XEasyPdfTemplateHandler.Text.build().setText(i + "-" + j);
        // 添加文本
        tableCell.addComponent(text);
        // 添加单元格
        tableRow.addCell(tableCell);
    }
    // 添加表格行
    tableBody.addRow(tableRow);
}
// 添加表格体
table.setBody(tableBody);
// 添加表格
page.addBodyComponent(table);
// 添加页面
document.addPage(page);
// 转换pdf
document.transform(outputPath);
```

#### 合并单元格

> ##### 对象模式 java 代码
```
// 定义输出路径
String outputPath = "E:\\pdf\\test\\fo\\test.pdf";
// 创建文档
XEasyPdfTemplateDocument document = XEasyPdfTemplateHandler.Document.build();
// 创建页面
XEasyPdfTemplatePage page = XEasyPdfTemplateHandler.Page.build();
// 创建表格
XEasyPdfTemplateTable table = XEasyPdfTemplateHandler.Table.build();
// 创建表格体
XEasyPdfTemplateTableBody tableBody = XEasyPdfTemplateHandler.Table.Body.build();
// 循环创建表格行
for (int i = 1; i < 4; i++) {
    // 创建表格行
    XEasyPdfTemplateTableRow tableRow = XEasyPdfTemplateHandler.Table.Row.build();
    // 循环创建表格单元格
    for (int j = 1; j < 4; j++) {
        // 创建表格单元格
        XEasyPdfTemplateTableCell tableCell = XEasyPdfTemplateHandler.Table.Cell.build().setBorderStyle("solid");
        // 创建文本
        XEasyPdfTemplateText text = XEasyPdfTemplateHandler.Text.build().setText(i + "-" + j);
        // 添加文本
        tableCell.addComponent(text);
        // 第一行第一列合并单元格
        if (i == 1 && j == 1) {
            // 合并三行
            tableCell.setRowSpan(3);
            // 添加单元格
            tableRow.addCell(tableCell);
        }
        // 第二行第二列合并单元格
        if (i == 2 && j == 2) {
            // 合并两列
            tableCell.setColumnSpan(2);
            // 添加单元格
            tableRow.addCell(tableCell);
            break;
        }else {
            if (j > 1) {
                // 添加单元格
                tableRow.addCell(tableCell);
            }
        }
    }
    // 添加表格行
    tableBody.addRow(tableRow);
}
// 添加表格体
table.setBody(tableBody);
// 添加表格
page.addBodyComponent(table);
// 添加页面
document.addPage(page);
// 转换pdf
document.transform(outputPath);
```

#### 设置项
- 设置边距：setMarginXXX
- 设置填充：setPaddingXXX
- 设置id：setId
- 设置宽度：setWidth
- 设置高度：setHeight
- 设置边框：setBorder
- 设置边框样式：setBorderStyle
- 设置边框折叠：setBorderCollapse
- 设置边框间距：setBorderSpacing
- 设置文本语言：setLanguage
- 设置行间距：setLeading
- 设置字符间距：setLetterSpacing
- 设置单词间距：setWordSpacing
- 设置空白空间：setWhiteSpace
- 设置字体名称：setFontFamily
- 设置字体样式：setFontStyle
- 设置字体重量：setFontWeight
- 设置字体大小：setFontSize
- 设置字体大小调整：setFontSizeAdjust
- 设置字体颜色：setFontColor
- 设置文本水平样式：setHorizontalStyle
- 设置文本垂直样式：setVerticalStyle
- 设置最小行高：setMinRowHeight
- 设置最小列宽：setMinColumnWidth
- 设置分页符-前：setBreakBefore
- 设置分页符-后：setBreakAfter
- 开启分页时保持：enableKeepTogether
- 开启分页时与上一个元素保持：enableKeepWithPrevious
- 开启分页时与下一个元素保持：enableKeepWithNext
- 开启自动省略表头：enableAutoOmitHeader
- 开启自动省略表尾：enableAutoOmitFooter
