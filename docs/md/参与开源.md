![fork项目](https://oscimg.oschina.net/oscnet/up-9003136b0122200429a7b4478fa43d27bb4.png "fork")

#### 2. Clone 项目

> <font size=3>在自己的仓库中，将刚刚 fork 的 x-easypdf clone 到本地</font>
![fork项目](https://oscimg.oschina.net/oscnet/up-9003136b0122200429a7b4478fa43d27bb4.png "fork")
